# knee-osteoarthritis-disease-detection-using-efficientnetB0

Knee Osteoarthritis Detection using EfficientNet-B0

This project presents a lightweight deep learning framework for automatic detection and classification of knee osteoarthritis (OA) from X-ray images. The model is based on EfficientNet-B0 and categorizes radiographs into three clinically meaningful classes: Healthy, Moderate, and Severe.

The workflow includes data preprocessing, class balancing, and transfer learning to improve classification performance and generalization. The proposed approach achieves high accuracy while maintaining low computational complexity, making it suitable for real-world clinical and edge deployment.


Key Features:

1.EfficientNet-B0 based lightweight model

2.Three-class OA severity classification

3.Data augmentation and class balancing

4.High accuracy with efficient computation
